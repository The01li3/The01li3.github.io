Scope, Hoisting & this Keyword

Scope

Global, Block or Function scopes.
Both const and let are block scoped, var is not block scoped, but try not to use.
Assigning a value to a variable that has not been declared automatically makes that variable global.
Variables start when declared, and close either in their block, their function, or when global when the tab is closed.

Hoisting

Variables can be declared after they have been used because of hoisting.
All declarations get moved to the top of the scope in which they have been created.
If assigned at declaring, the assigning doesn't also get moved.

Let and const get hoisted, but not initialized. Honestly just hoist them yourself, it makes code more readable.

For the most part, ignore JS's behaviour here and declare variables at the top, its just better

this

this, refers to an object, you cannot change the value of this, it a keyword.

Object Method

Refers to the objects values.

const person = {
    firstName: "John",
    lastName: "Doe",
    id: 5566,
    fullName : function() {
        return this.firstName + " " + this.lastName; //person.firstName...
    }
};

Alone

Alone, this refers to the global object.

let x = this; //[object Window]

Function

In a non strict function, this refers to the global object.

function myFunction() {
    return this;
}

Event Handlers

In HTML event handlers, this refers to the HTML element that received the event:

<button id="btn-properties-eg" type="button" class="btn btn-show" onclick='fnShowHide(this,"properties-eg")' >Show # Property Examples</button%gt

Call and Apply

Both allow you to call a function using different parameters.
Call is comma, Apply is an array.
The difference is that apply lets you invoke the function with arguments as an array; call requires the parameters be listed explicitly. 

function Product(name, price) {
    this.name = name;
    this.price = price;
}

function Food(name, price) {
    Product.call(this, name, price);
    this.category = 'food';
}

console.log(new Food('cheese', 5).name); // Expected output: "cheese"
const numbers = [5, 6, 2, 3, 7];
const max = Math.max.apply(null, numbers);
