Methods

Not likely to use a lot, so some basics:

Method Desc
x.length Length of string X
x.at(i) Get the ith+1 character of x
x.slice(i, j) Get string characters from i to j, "j" can be empty to show the rest
x.subtring(i, j) If empty, treated as 0
x.substr(i, j) Same as above but j is count amount of characters
x.toUpperCase() All of x to upper case
x.toLowerCase() All of x to lower case
x.trim() Removes white space
x.replace("a","b") Replaces A with B, case sensitive unless you use /a/i
x.replaceAll("a","b") Replaces all A with B, case sensitive unless you use /a/i
x.split(",") Splits into an array based off ","

String Searching

Not likely to use a lot, so some basics for "bob likes to bob in the sea" (check console log for more details):

  • x.indexOf("bob") - 0
  • x.lastIndexOf("bob") - 13
  • x.search("bob") - 0
  • x.match("bob") - ["bob"]
  • x.matchAll("bob") - bob,bob
  • x.includes("bob") - true
  • x.startsWith("bob") - true
  • x.endsWith("bob") - false

Template Literals

Can use backticks to use both quotes `It's name is "Fido"`
${...} with backticks ` means you can use it without "+" e.g: 

let firstName = "John";
let lastName = "Doe";
let text = `Welcome ${firstName} ${lastName}!`;

vs

let firstName = "John";
let lastName = "Doe";
let text = "Welcome" + firstName + lastName + "!";

In the first example "text" becomes a "Template Literal", and the variables are using "Interpolation", works well for HTML input. Particularly looping content, lists, tables etc. e.g.

let header = "Templates Literals";
let tags = ["template literals", "javascript", "es6"];

let html = `<h2>${header}</h2><ul>`;
for (const x of tags) {
    html += `<li>${x}</li>`;
}
html += `</ul>`;
