Modules

Modules allow you to break up the code into separate files

<script type="module">
    import { name, age } from "./person.js";
</script>

Can then have another file with the export of the variables like so:

const name = "Jesse";
const age = 40;

export {name, age};

You can also have just default exports from files that would work:

import message from "./message.js";

Where the export file would look like:

const message = () => {
    const name = "Jesse";
    const age = 40;
    return name + ' is ' + age + 'years old.';
};

export default message;
