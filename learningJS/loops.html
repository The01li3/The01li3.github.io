Loops

For Loop

Fairly standard:

for (let i = 0; i < x.length; i++) {
    text += x[i] + "
";
}

Use let to keep it inside the block.

Expression 1

(One Time Run; Condition For The Block; Executed Everytime AFTER Block).
In theory you could have the "One Time Run" to contain multiple expressions.
Can also omit expression 1 completely with just a blank then ; which could be useful if the loop parameter is defined previously.

Expression 2

Can be left out, just make sure the loop always ends in a break to avoid recursion and browser crashing.

Expression 3

Also optional, could increment in the block, could increment multiple amounts etc.

For In

Can use a few loops for looping through arrays.

for (let i in x) {
    text += x[i];
}

The in knows to loop over the object, and will return a the key of i everytime.

But it'll likely be better to use a x.forEach(myFunction) call to call a function for every iteration.

For Of

for (let x of cars) {
    text += x;
}

Similar to the for in, but instead of giving a pointer to the value, it gives the value:

for...in

text += x[i]

for...of

text += x

While

Loops the below, while the condition is true.

while (i $lt 10){
    text = x[i];
    i++
}

Do While works the same way, but it'll always run through it once before getting to the check.
So works a similar way to the for, but the conditions in the for brackets are normally placed inside the loop.

Break

Break will "jump out" of the current loop.
Continue will "jump over" to the next iteration of the loop.
Can also use a "label" to break out of a specific block of code, if that block has been labelled.

Iterables

Good example to use a for...of loop to demonstrate.
It'll iterate over x in different ways depending on what x is.
Strings get each character looped.
Arrays get each element looped.
Maps get each set looped.
Home made iterables: 

function myNumbers() {
    let n = 0;
    return {
        next: function() {
            n += 10;
            return {value:n, done:false};
        }
    };
}

// Create Iterable
const n = myNumbers();
n.next(); // Returns 10
n.next(); // Returns 20
n.next(); // Returns 30
