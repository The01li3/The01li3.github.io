? and : can be used as an "else" type of situation variablename = (condition) ? value1:value2

if, else, elseif, switch, all self explanitory.

if (time < 10) {
    greeting = "Good morning";
} else if (time < 20) {
    greeting = "Good day";
} else {
    greeting = "Good evening";
}

Same as a normal case statement, uses === scrict comparison.

switch (variable) {
default:
    message = "Not A,B,C,D"
    break; //Doesnt have to be last
case A:
    message = "A";
    break;
case B: 
    message = "B";
    break;
case C:
case D:
    message = "C or D"; //can share case blocks, if this or this

}
