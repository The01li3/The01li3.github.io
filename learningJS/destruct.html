Destructuring

Objects

Can make objects properties into variables, give them an alias, and add new variables

const person = {
    firstName = "bob",
    lastName = "bobberson"
};

let {firstName : first, lastName : last, country = "Nunya"} = person;

Strings

Strings get the characters.

let vName = "Oliver";
let [c1,c2,c3,c4] = name; //c1 = O, c2 = l... etc.

Arrays

Works in much the same way as strings, but with arrays. You can skip elements in the array, or pick positions. Can end with a rest property staring remaining in a new array

let [array1, array2, array3,,,array6, ...somemore] = arrayX
let {[0]:array1 ,[1]:array2} = arrayX;

Swapping

You can swap 2 variables, without the need for a 3rd temp variable.

[firstName, lastName] = [lastName, firstName];
