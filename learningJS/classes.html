Classes

Intro

Classes are templates for JS objects.
Use class to create the class.
Then also need to have a constructor within the class:

class Car {
    constructor(name, year) {
        this.name = name;
        this.year = year;                
    }
    age() {
        const date = new Date();
        return date.getFullYear() - this.year;
    }
    static hello(x) {
        return "Hello " + x.name;
    }
}

You can then create objects with the class:

const vCar1 = new Car("Renault", "2006");

This will make vCar1.name, vCar1.year and vCar1.age() able to be used.
You can use Getters and Setters as in objects if need be.
Class declarations are not hoisted, so declare BEFORE use. You can use static in constructors meaning you cannot access them through the object, but just directly. 

Car.hello(); //Would return "Hello" if there was no x input parameter
                    vCar1.hello(); //Will raise an error
                    Car.hello(vCar1);//Will return "Hello Renault"

Inheritance

To use class inheritance you can use extends, a class created with class inheritance inherits the methods from another class (think progress "LIKE").
You can then give it a constructor of super(propertyName) or super.propertyName to access the inherited classes methods. 

class Foo {
constructor(name) {
    this.name = name;
}

getNameSeparator() {
    return '-';
}
}

class FooBar extends Foo {
constructor(name, index) {
    super(name);
    this.index = index;
}

getFullName() {
    return this.name + super.getNameSeparator() + this.index;
}
}

const firstFooBar = new FooBar('foo', 1);

console.log(firstFooBar.name);
// Expected output: "foo"

console.log(firstFooBar.getFullName());
// Expected output: "foo-1"
