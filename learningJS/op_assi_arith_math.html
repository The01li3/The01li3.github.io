Operators, Assignment, Arithmetic & Math

Operators

Fairly standard: +, -, *, **(Exponentiation), /, %, ++, --

Assignments

There are the normal JS operators, bitwise, shift and logical assignments (shorthand += etc.).

JS

Same as the normal operators: =, +=, -=, *=, /=, %=, **=
Then just being the same as =, x+y, x-y... etc.

Shift

Uses binary to shift the values over: <<=, >>=, >>>=
So 4 = 1 << 2 as 00000001 moved over 2 is 00000100

Bitwise

I dont really get these but: &=, ^=, |=

Logical

if X then Y &&=, ||=, ??=
&&= if first value is TRUE, then second value is assigned
||= if first value is FALSE, then second value is assigned
??= if first value is undefined or null, then second value is assigned
