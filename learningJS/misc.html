Regular Expressions

Can be used to search for text (think /n etc in notepad) with different modifiers.
i is used for case insensitive for example:

x.search(/bob/i);

My name is Bob, not bob in the sea. Just Bob. Not boob. 123456

Expression Desc Example Output
i Perform case-insensitive matching x.match(/JUST/i); just
g Perform a global match (find all) x.match(/Bob/gi); Bob,bob,Bob
m Perform multiline matching x.match(/Bob/m); Bob
[abc] Find any of the characters between the brackets x.match(/[o]/g); o,o,o,o,o,o,o
[0-9] Find any of the digits between the brackets x.match(/[1-2]/g); 1,2
(x|y) Find any of the alternatives separated with | x.match(/(bob|my)/gi); My,Bob,bob,Bob
\d Find a Digit x.match(/\d/g); 1,2,3,4,5,6
\s Find a whitespace character x.match(/\s/g); , , , , , , , , , , ,
\b Find a match at the beginning of a word like this: \bWORD, or at the end of a word like this: WORD\b x.match(/\bju/gi); Ju
\uxxxx Find the Unicode character specified by the hexadecimal number xxxx x.match(/\u0042/gi); B,b,b,b,B,b,b
n+ Matches any string that contains at least one n x.match(/e+/gi); e,e,e
n* Matches any string that contains zero or more occurrences of n, search for all "b" followed by one or more "o" x.match(/bo*/g); b,bo,b,b,boo,b
n? Matches any string that contains zero or one occurrences of n x.match(/bo?/g); b,bo,b,b,bo,b

Can use RegExp Methods:

/e/i.test("The best things in life are free!"); //will return true, since there is an e

Can also use exec to return the value as an object.

JSON

JavaScript Oject Notation.
Dealt with JSON before, its the normal return of API calls, basically an array of objects. 

"employees":[{
        "firstName":"John",

    },{
