Dates

Basics

Created with d = new Date(), different parameters can be added for days, months seconds etc. Can display with d.toString() d.toDateString() or d.toISOString()

Formats

Pretty much just exists 3 formats, ISO, short and long.
Always try and use full date and times, 0's included e.g. 2024/03/11

Methods

Luckily fairly self explanatory, don't use getYear() its old and only returns 2 digit year.

  • x.getFullYear()
  • x.getMonth()
  • x.getDate()
  • x.getDay()
  • x.getHours()
  • x.getMinutes()
  • x.getSeconds()
  • x.getMilliseconds()
  • x.getTime()

Can use the same methods but "set" to set the parameters in the variable
