AJAX

Asynchronous Javascript And XML.
RMS used jQuery for this. with $.ajax({url:"",data:"",etc:etc}).
But normal JS uses new XMLHttpRequest.

  • Create the object
  • Define the callback function
  • Open the object
  • Send a request
const myAJAX = new XMLHttpRequest();
myAJAX.onload = function() {
    document.getElementById("demo").innerHTML = this.responseText;
}

myAJAX.open("GET",URL,true); // true/false is async yes/no, default is true
myAJAX.send();

Data can some back as a responseText or responseXML
Can use that data coming back to display in HTML, template literals are a nice way to do it.
